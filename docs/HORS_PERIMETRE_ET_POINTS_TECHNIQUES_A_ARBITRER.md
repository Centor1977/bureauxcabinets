# Hors périmètre et points techniques à arbitrer

## Hors périmètre V0

Ne pas développer maintenant :

- espace client ;
- espace installateur ;
- back-office complet ;
- paiement en ligne ;
- matching automatique ;
- calendrier synchronisé ;
- devis automatique ;
- compte utilisateur ;
- système d’avis ;
- blog ;
- pages locales multiples ;
- parcours admin / artisan complet ;
- système de relance.

## Points techniques à arbitrer avec le développeur

Ce pack ne verrouille pas :

- l’outil ou framework de développement ;
- la base de données ;
- le stockage photo ;
- la destination des leads ;
- les notifications ;
- les emails automatiques ;
- l’hébergement ;
- le tracking ;
- le système cookies ;
- l’anti-spam ;
- la configuration de production.

Claude Code doit signaler ces points comme arbitrages, pas les décider seul.
