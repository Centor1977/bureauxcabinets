## Direction artistique verrouillée

À respecter :

- premium ;
- locale ;
- sobre ;
- crédible terrain ;
- immobilier / sécurité / logement réel ;
- bleu nuit ;
- blanc cassé ;
- gris chaud clair ;
- accent teal ;
- beaucoup d’espace blanc ;
- visuels de portes, façades lilloises, interphones, logements meublés ;
- iconographie limitée et sobre.

À éviter :

- effet SaaS générique ;
- effet annuaire travaux ;
- illustrations startup ;
- surabondance d’icônes ;
- images américaines ou trop stock ;
- faux avis ;
- étoiles ;
- chiffres non prouvés.

## Signaux locaux et Google Maps

Le site doit soutenir la visibilité locale Google Maps / Google Business Profile sans bourrage SEO.

À intégrer naturellement :

- **Accès Autonome Lille** ;
- **Lille & métropole** ;
- **logements meublés** ;
- **locations de courte durée** ;
- **biens gérés à distance** ;
- **serrure connectée** ;
- **interphone connecté** ;
- **code** ;
- **badge** ;
- **ouverture à distance**.

Footer exact :

**Accès Autonome Lille — Solutions d’accès autonome pour logements meublés, locations de courte durée et biens gérés à distance à Lille métropole.**
