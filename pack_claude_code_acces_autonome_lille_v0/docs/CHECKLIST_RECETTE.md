L’ordre des sections est respecté :

1. Header
2. Hero avec formulaire étape 1 visible

5. Solutions
6. Qualification

12. Footer localisé

## Formulaire

- Le formulaire est en 3 étapes.
- En hero, seule l’étape 1 est visible.
- L’étape 1 contient uniquement : Type de bien, problème à résoudre, accès concerné, ville / code postal, délai souhaité.
- “Nombre de logements” n’apparaît pas en étape 1.
- Les étapes 2 et 3 sont annoncées sous forme de lignes fermées.
- Les champs multi-réponses fonctionnent.
- Les options “Je ne sais pas” sont exclusives.
- La photo est facultative.
- L’utilisateur peut supprimer une photo avant envoi.
- Les créneaux sont présentés comme disponibilités de rappel, jamais comme rendez-vous confirmé.
- La mention obligatoire sur les créneaux est visible.
- Le consentement est visible avant soumission finale.
